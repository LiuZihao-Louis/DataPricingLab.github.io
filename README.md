# 第二周研究材料

**题目：** 零售行业数据资产估值与收益分配研究——以全渠道智能补货为例

## 内容

- `docs/`：研究摘要、研究进展报告、估值与收益分配原型（DOCX 和 PDF）。
- `data/`：案例卡片和证据库（XLSX）。
- `research/`：第三周验证任务和参考资料索引。
- 根目录 `index.html`：静态网页原型，可用于 GitHub Pages。

## 研究边界

本项目讨论经过授权、治理和场景化处理后的数据服务。未公开的价格、合同条款和分成比例均不作为事实使用；相关内容在材料中标记为研究假设或待核验问题。

## 使用方式

1. 阅读 `docs/01_Week2_Research_Progress_Report.pdf`。
2. 使用 `data/` 中的案例卡片和证据库追溯结论。
3. 打开根目录 `index.html` 查看网页原型。
4. 依据 `research/06_Week3_Verification_Plan.md` 继续补充价格和参数证据。
